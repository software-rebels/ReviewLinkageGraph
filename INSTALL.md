## Required tools and packages
Our replicaion package is comprised of Python and R programs. To replicate our study, install:
- Python 3.6
- R 3.3.2

To execute RUN.sh, install the following libraries:

[Python]
- numpy
- nltk
- sklearn
- imblearn
- pandas
- scipy
- itertools
- pymongo
- locale

[R]
- ggplot2
- scales
- tidyverse
- reshape2
